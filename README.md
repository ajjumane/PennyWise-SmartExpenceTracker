# 💰 PennyWise: Smart Expense Tracker

**PennyWise** is a high-performance, premium web-based expense tracker designed for professional personal finance management. It features a stunning glassmorphism UI, real-time data visualization, and an **AI-powered Receipt Scanner**.

## 🚀 Key Features

- **🏠 Premium Dashboard**: A real-time overview of your monthly spending vs. budgets.
- **📸 AI Receipt Scanner**: Upload any bill and let the integrated **OCR (Tesseract.js)** automatically identify items, descriptions, and amounts.
- **🍱 Multi-Item Support**: The scanner can break down a single receipt into multiple individual line items for granular tracking.
- **🏗️ Smart Categorization**: Automatic classification of merchants (e.g., Starbucks -> Food, Uber -> Travel).
- **🔒 Privacy-Focused**: No server-side processing—all OCR and data storage (LocalStorage) happen entirely within your browser.
- **📊 Interactive Visualization**: Dynamic doughnut charts (Chart.js) to visualize spending breakdowns per category.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Styling**: Tailwind CSS, Custom Glassmorphism CSS
- **AI/OCR**: Tesseract.js
- **Charts**: Chart.js
- **Icons**: SVG, Flaticon

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ajjumane/PennyWise-SmartExpenceTracker.git
   ```
2. **Open the App**:
   Simply open `auth.html` in any modern web browser to log in or create an account.

## 📝 Multi-Item Scanner Workflow
1. Navigate to the **Scan Receipt** tab.
2. Upload a clear bill image.
3. Review and edit the detected items in the dynamic list.
4. Set a category and click **Add Selected Expenses**.
