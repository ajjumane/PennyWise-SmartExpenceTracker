# 💰 PennyWise - Smart Expense Tracker

PennyWise is an intelligent expense tracking application designed to simplify personal finance management using automation and AI-powered features.

---

## 🚀 Features

- 📅 Track daily expenses by date and categories  
- 🎯 Set and manage monthly budget limits  
- 🔁 Add automatic fixed expenses (rent, subscriptions, etc.)  
- 🎤 Add transactions using voice commands  
- 🧾 AI-based receipt scanner to extract and log expenses automatically  
- 📊 Visual spending insights with category-wise pie charts  
- 🔐 Secure user authentication system  

---

## 🧠 Core Idea

PennyWise focuses on reducing manual effort in expense tracking by integrating voice input and AI-powered receipt scanning, making financial management faster and more intuitive.

---

## ⚙️ Tech Stack

- Backend: Flask / Django  
- Frontend: HTML, CSS, JavaScript  
- Database: SQLite / MySQL  
- AI Features: OCR / Image Processing  
- Authentication: Secure login system  

---

## 📈 Key Highlights

- Combines automation + AI in finance tracking  
- Reduces manual data entry  
- Provides clear spending analysis  
- Designed for real-world usability
